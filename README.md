# LangChain Ollama Chatbot

A conversational chatbot built using LangChain and Ollama with local LLMs.
Supports multi-turn conversations with message history persistence.

## Features
- Uses Ollama (local LLM, no paid APIs)
- Multi-turn chat with memory
- Chat history stored in JSON
- Built with LangChain Runnable interfaces

## Requirements
- Python 3.10+
- Ollama installed and running
- Dependencies listed in requirements file

## How to Run
1. Install dependencies
   pip install -r custom_gpt_reqs.txt
2. Start Ollama
3. Run the notebook

## Tech Stack
- Python
- LangChain
- Ollama
- Local LLM (Mistral)
